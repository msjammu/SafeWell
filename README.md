# SafeWell

A borewell-safety platform to **prevent** children from falling into uncovered borewells,
**detect** falls early, and **coordinate** rescue.

Motivated by recurring tragedies such as the [Ambala borewell incident](https://www.tribuneindia.com/news/haryana/4-year-old-boy-falls-into-borewell-at-ambala-village-rescue-operation-on/),
where a 4-year-old fell into a 350 ft uncovered borewell.

## Pillars
1. **Prevent** — geo-tagged well registry + public reporting + owner accountability.
2. **Detect** — low-cost IoT smart-cap with fall/intrusion alerts (later phase).
3. **Respond** — rescue-coordination dashboard and live telemetry (later phase).
